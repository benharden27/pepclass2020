    R Resources

    Learning any programming language can be both completely frustrating and hugely rewarding. Be kind to yourself and try to see past the struggles to the glorious highs that will come.

    We’ll be doing some intro work in the course, mostly centered around data visualization, but here is a selection of other resources that you might find useful in exploring the field further.

    R for Data Science by Garrett Grolemund and Hadley Wickham: Core text on exploring data in R. All of it is worth a look-in.

    ggplot2: Elegant Graphics for Data Analysis by Hadley Wickham: A ton of information about how ggplot works and how you can extend it to suit your needs.

    Data Visualization: A practical introduction by Kieran Healy: A really nice text that combines the why and how of data visualization

    RStudio Primers: Are great intros to a bunch of things you might want to do in R

    Rstudio Remote Learning: A great selection of online resources to help folks learning R

    Data Visualization Class by Dr. Andrew Heiss: A really nice class offering with a ton of great resources for learning R for data work.
